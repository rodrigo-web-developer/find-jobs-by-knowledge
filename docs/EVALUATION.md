# INTRODUCTION

User can create assessment tests by selecting tags, the system will then generate a test with questions related to those tags, at least 5 questions per level. The user can then answer the questions and the system will use the answers to determine the user's skill level for each tag and recommend jobs that match their profile.

## Questionary

Questionary will be generated and saved based on the tags selected by the user. The system will select questions from the question database that are related to the selected tags. The questionary will be saved in the database and can be accessed by the user at any time.

## Results

The results of the assessment will be calculated based on the user's answers to the questions. The system will determine the user's skill level for each tag based on the percentage of correct answers for each level. The results will be saved in the database and can be accessed by the user at any time. The system will also use the results to recommend jobs that match the user's skill level and preferences.
